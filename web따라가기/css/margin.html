margin - 요소의 외부 여백

margin의 방향별 지정

  • margin-top: n ;
  • margin-right: n ;
  • margin-bottom: n ;
  • margin-left: n ;

margin의 값 갯수별 지정

  • margin: n ; 양사방을 동일하게
  • margin: n n;상하 좌우 순으로 동일
    항상 수평 중앙에 위치 = margin: n auto;
  • margin: n n n ; 상 좌우 하 순으로
  • margin: n n n n ; 상 우 하 좌 순으로 각각
