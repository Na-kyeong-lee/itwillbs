background 배경

  1. background-color: ; 배경색상
  2. background-image:url("경로"); 배경이미지
  3. 배경이미지 2개 적용하려면 url("경로A"), url("경로B")
    식으로 , 를 추가하며 나머지 속성들도 값을 , 로 추가함

background-image의 속성들

  • background-repeat: ; 반복 지정
    • repeat 기본값. 양사방 반복
    • repeat-x 가로로만 반복
    • repeat-y 세로로만 반복
    • no-repeat 반복 안함
  • background-size: ; 크기 지정
    • contain 이미지를 가로나 세로에 최대한 맞추기 (이미지가 잘려나가지 않음)
    • cover 이미지 하나로 전체 다 덮기 (이미지가 잘려나갈 수 있음)
    • n n 가로 세로의 순으로 직접 지정
  • background-position: ; 위치 지정
    • left/center/right top/center/bottom
    • n n 가로 세로의 순으로 직접 지정
  • background-attachment: ; 스크롤 여부 지정
    • scroll
    • fixed
    • local

background-attachment: ; 스크롤 여부

  1. background-attachment:scroll;
    자신의 스크롤바에는 고정, 브라우저 스크롤바에는 스크롤.
  2. background-attachment:fixed;
    자신의 스크롤바와 브라우저 스크롤바에 모두 고정.
  3. background-attachment:local;
    자신의 스크롤바와 브라우저 스크롤바에 모두 스크롤.
