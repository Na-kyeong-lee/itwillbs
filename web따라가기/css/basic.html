CSS구문

선택자 { 속성:속성값; }

CSS 적용방식 3가지

  • External방식 - html문서와 별개의 외부파일(*.css)에 스타일소스를 적용하고 연결시킴.
    html문서안의 head내에 <link href="*.css" rel="stylesheet">
    유지보수에 유리하며 통일된 스타일로 사이트 구축 가능.
  • Internal방식 - html문서안의 head내에 스타일소스를 적용.
    <style type="text/css"> ... </style> 해당 html문서에만 스타일이 적용 됨.
  • Inline방식 - tag 뒤에 바로 스타일소스를 적용. <tag style="속성:속성값;">
    소스가 길어지고 유지보수에 불리하며 통일된 스타일로 사이트 구축이 불가능.
    우선 순위가 가장 높음. 본문(body)만 수정 가능한 작업환경일 때 사용

CSS선택자 대표 3가지

  • tag선택자 - tag명 그대로 선택자로 활용.
    자동 적용, 강제 적용
  • id선택자 - tag 뒤에 id를 추가하여 선택자(#id값)로 활용.
    한번만 적용됨. 주로 구조적인 용도로 쓰임
  • class선택자 - tag 뒤에 class를 추가하여 선택자(.class값)로 활용.
    중복 적용 가능, class끼리 조합 가능. 주로 디자인적인 용도로 쓰임
