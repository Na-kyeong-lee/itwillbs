⛵List - 목록 만들기

html tag로 만드는 목록 3가지

<ul> = 순서가 없는 목록
<ol> = 순서가 있는 목록
<dl> = 정의 목록

ul - 순서가 없는 목록

  • <ul> = 순서가 없는 목록
  • <ol> = 순서가 있는 목록
  • <dl> = 정의 목록
  • html
  • css
  • js

문제. 순서 없는 목록 만들기

  • 빨강
  • 초록
  • 파랑

ol - 순서가 있는 목록

  1. red=rgb(255,0,0)=#ff0000
  2. green=rgb(0,255,0)=#00ff00
  3. blue=rgb(0,0,255)=#0000ff
  4. yellow=rgb(255,255,0)=#ffff00
  5. white=rgb(255,255,255)=#ffffff
  6. black=rgb(0,0,0)=#000000
  1. html - 웹페이지의 구조를 담당.
  2. css - 웹페이지의 디자인을 담당.
  3. js - 웹페이지의 동적인 효과를 담당.
