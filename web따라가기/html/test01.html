html 첫수업

html?

Hyper Text Markup Language.
웹페이지를 만드는 프로그램언어.
웹페이지의 구조를 담당.

tag의 특징

html문서는(웹페이지) tag, 명령(요소)으로 구성 됨.
tag는 일반적으로 각괄호 안에 들어가며,
시작tag와 종료tag의 한 쌍으로 구성 됨.
대소문자 구별은 없으나 소문자로 작성.

html문서의 필수tag

 <html>     ------ 웹페이지의 시작 선언
  <head>     ------ 머리말의 시작 선언
   <title> ...  </title>  ------ 제목(탭) 지정
  </head>    ------ 머리말의 끝 선언  
  <body>     ------ 본문의 시작 선언

  </body>     ------ 본문의 끝 선언
 </html>     ------ 웹페이지의 끝 선언
