html5에서 추가된 input type의 속성들

1. autofocus (autofocus="autofocus") 자동포커싱. 해당 요소가 대기상태

2. autocomplete="on/off" 자동완성 기능을 켜거나 끔

3. placeholder="***" 입력 예시나 설명 등을 추가


4. required(required="required") 필수입력요소로 지정

5. readonly (readonly="readonly") 읽기만 가능한 입력요소로 지정

클릭, 복사할 수 있고 제출할 때 전송 됨

6. disabled (disabled="disabled") 사용불가한 입력요소로 지정

비활성화. 클릭, 복사할 수 없고 제출할 때 전송되지 않음
