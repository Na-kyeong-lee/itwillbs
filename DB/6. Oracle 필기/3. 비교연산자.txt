<Like 비교연산자>
- % : 0개 또는 여러개의 문자가 올 수 있음.
- _ : 반드시 1개의 문자가 와야함.

a로 시작되는 문자열 : 'a%'
a가 포함된 문자열 : '%a%'
a로 끝나는 문자열 : '%a'
두번째 문자가 a인 문자열 : '_a%'
끝에서 세번째 문자가 a인 문자열 : '%a__'

<비교연산자>
=			<-->	<>, !=
>, >=			<-->	<, <=
between A and B		<-->  	not between A and B
in : (=, OR)		<-->	not in : (<>, AND)
like			<-->	not like
is null			<--> 	is not null
